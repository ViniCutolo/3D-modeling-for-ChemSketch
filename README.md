This is an independent project and it has no affiliation with ACD/Labs. ChemSketch is a software tool developed by ACD/Labs.

The idea of this project is to read an .mol file from ChemSketch and create a 3D model that you can rotate and visualize. 
An example molecule (trinitrotoluene) is included in the repository for demonstration purposes.

## Dependencies
This project uses the Python standard library and the VPython library for 3D molecular visualization.
VPython is distributed under a BSD-style license. Please refer to the VPython project for licensing details.
